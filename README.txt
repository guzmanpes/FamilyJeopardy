FAMILY JEOPARDY — MEDIA / QUESTION / MEDIA / ANSWER

Each clue supports:
1. Optional media before the question
2. Written question
3. Optional media after the question
4. Hidden written answer

Either media stage can be set to No media.

Supported media:
- YouTube or MP4 video
- MP3 audio
- Images

For GitHub-hosted files, put media in the media folder and use paths such as:
media/intro.mp3
media/video.mp4
media/photo.jpg
