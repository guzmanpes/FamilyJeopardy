Each clue can follow: media → question → media → answer. Both media fields are optional, so a clue can also be question → answer, media → question → answer, or question → media → answer.

Supported media: YouTube, MP4 video, MP3 audio, and images. When Video is selected, optional Start and End fields appear. Enter seconds such as 83, or time formats such as 1:23 and 1:40. A timestamp already included in a YouTube link is also recognized.

For files uploaded to GitHub, use paths such as media/intro.mp3, media/clip.mp4, or media/photo.jpg.
